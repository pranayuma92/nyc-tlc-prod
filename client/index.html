API Documentation

Version 1.0 | Last Updated: November 2024

Introduction

This API allows you to access data related to Yellow Taxi Trip Data.

Base URL

https://your-domain.com/api/v1

Endpoints

  • POST /trip

    Returns a list of trips data

    Request payload example 
    {
    "dropOffTime: "2014-12-31",
    "pickUpTime": "2014-12-31",
    "fareAmount": "",
    "distance": "",
    "paymentType": "CRD",
    "sort": "desc",
    "limit": 25,
    "offset": 0
}
